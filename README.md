# 📖 Kindle 自動スクリーンショットツール

pyautoguiを使ってKindle書籍を自動でスクリーンショットするPythonツールです。

## 🚀 セットアップ

### 必要なパッケージ
```bash
pip install pyautogui Pillow img2pdf tqdm
```

## 📋 使い方

### ツールの起動
```bash
python kindle_screenshot.py
```

### 基本的な手順

1. **座標測定** — メニューから `[2]` を選んで、キャプチャ範囲の座標を測定
2. **設定確認** — メニューから `[5]` で設定を確認（必要なら `[3]` で変更）
3. **スクショ開始** — メニューから `[1]` を選んで開始
4. **PDF生成** — スクショ完了後、自動的にPDF生成するか確認されます

### メニュー一覧

| メニュー | 機能 |
|---------|------|
| `[1]` 📸 | スクリーンショットを開始する |
| `[2]` 🖱️ | キャプチャ座標を測定する |
| `[3]` ⚙️ | 設定を変更する |
| `[4]` 📄 | 既存画像からPDFを生成する |
| `[5]` 📋 | 現在の設定を表示する |
| `[6]` 🚪 | 終了 |

## ⚙️ 設定項目

設定は `kindle_config.json` に保存されます。

| 設定項目 | 説明 | デフォルト |
|---------|------|-----------|
| `page` | ページ数 (0=自動停止モード) | `0` |
| `x1`, `y1` | 取得範囲の左上座標 | `0, 0` |
| `x2`, `y2` | 取得範囲の右下座標 | `1920, 1080` |
| `span` | スクショ間隔 (秒) | `0.8` |
| `wait_before_start` | 開始前の待機時間 (秒) | `5` |
| `image_format` | 画像形式 (png/jpg) | `png` |
| `jpg_quality` | JPG品質 (1-100) | `90` |
| `resize_width` | リサイズ横幅 (0=原寸) | `0` |
| `auto_stop` | 重複検出で自動停止 | `true` |
| `auto_stop_threshold` | 何回連続同一画像で停止するか | `3` |
| `generate_pdf` | 完了後にPDF生成するか | `true` |
| `page_direction` | ページ送りキー (right/left) | `right` |

## ✨ 機能一覧

### 基本機能
- 📸 指定範囲の自動スクリーンショット
- ⌨️ 自動ページ送り（右/左キー対応）

### 追加機能
- 🖱️ **座標測定ヘルパー** — マウスカーソルで座標を簡単測定
- 🔍 **重複検出・自動停止** — ページが変わらなくなったら自動停止（ページ数不明時に便利）
- 📄 **PDF変換** — スクショをまとめてPDFに変換
- 📊 **プログレスバー** — 進捗状況をリアルタイム表示
- ⚙️ **設定ファイル** — 設定をJSONで保存・再利用
- 🖼️ **画像リサイズ** — 保存時に自動リサイズ
- 🎨 **PNG/JPG対応** — 用途に合わせて画像形式を選択
- 🛡️ **安全機能** — マウスを画面左上に移動で緊急停止 (pyautogui FAILSAFE)

## ⚠️ 注意事項

- Kindleアプリをフルスクリーンまたはウィンドウモードで表示した状態で使用してください
- **`Ctrl+C`** でスクショを中途中断できます
- **マウスを画面左上隅に移動** すると緊急停止します
- 本ツールは個人利用を目的としています。著作権にご注意ください
